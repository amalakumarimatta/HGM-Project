# 🧠 Hybrid Graph Mamba -- Polyp Segmentation

## 📌 Overview

This project implements a deep learning-based medical image segmentation model for detecting and segmenting polyps in colonoscopy images using the **Hybrid Graph Mamba architecture**.

The model processes medical images and highlights polyp regions to support early diagnosis and improve colorectal disease detection.

---

## 🚀 Key Features

- 🔬 Polyp segmentation using Hybrid Graph + Mamba architecture  
- ⚡ GPU-accelerated training and inference (Kaggle environment)  
- 📊 Evaluation using medical imaging metrics  
- 🖼️ Visualization of segmentation results  

---

## 🛠️ Tech Stack

- Python  
- PyTorch  
- Torch-Geometric  
- OpenCV  
- NumPy  
- Kaggle Notebook Environment  

---

## 📊 Dataset

Used standard polyp segmentation datasets:

- CVC-300  
- CVC-ClinicDB  
- CVC-ColonDB  
- ETIS-LaribPolypDB  
- Kvasir
